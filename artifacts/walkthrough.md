# Insta-Lite 프로젝트 결과보고 (Walkthrough)

경량 인스타그램(Insta-Lite)의 핵심 기능을 모두 구현하고 검증을 완료했습니다.

## 구현된 주요 기능

### 1. 인증 시스템 (Auth)
- **회원가입**: 이메일 및 비밀번호를 통한 계정 생성 (bcrypt 암호화 적용).
- **로그인**: JWT(JSON Web Token) 발급을 통한 세션 유지.
- **보안**: 비밀번호 해싱 및 토큰 기반의 인가 시스템 구축.

### 2. 게시물 관리 (Post CRUD)
- **작성**: 이미지 파일 업로드(`Multipart/form-data`) 및 문구 작성.
- **조회**: 전체 게시물 최신순 피드 및 개별 상세 페이지 제공.
- **수정/삭제**: 작성자 본인만 가능하도록 권한 검증 로직 구현.

### 3. 소셜 기능 (Interaction)
- **좋아요**: 토글 방식의 좋아요 기능.
- **댓글**: 게시물 상세 페이지에서의 댓글 작성 및 실시간(갱신형) 조회.

### 4. 프론트엔드 (UI/UX)
- **바닐라 JS**: 외부 프레임워크 없이 `fetch` API와 DOM 조작으로 SPA 느낌의 구현.
- **반응형 디자인**: 모바일 및 데스크탑을 지원하는 카드형 레이아웃.

## 실행 방법

1. **서버 실행**:
   ```bash
   source .venv/bin/activate
   uv run uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
   ```

2. **접속**:
   - 웹 브라우저: `http://localhost:8000`
   - API 문서(Swagger): `http://localhost:8000/docs`

## 기술적 특이사항

- **Database**: `sqlite3.Row`를 사용하여 직관적인 필드 접근을 구현하고, Pydantic 모델 반환 시 `dict()` 변환을 통해 검증 오류를 방지했습니다.
- **Password Hashing**: Python 3.12 환경에서의 `passlib` 호환성 이슈를 고려하여 `bcrypt` 라이브러리를 직접 사용하는 견고한 방식으로 구현했습니다.
